### 1. Executive Summary (1~2페이지)

투자 의사결정에 필요한 핵심 정보를 한 눈에 파악할 수 있도록 구성.

**1.1 ETF 개요 테이블:**

| 항목 | 내용 |
|---|---|
| 티커 | {ticker} |
| 정식 명칭 | {name} |
| 운용사 | {issuer} |
| AUM | ${aum} |
| 보수율 | {ter}% |
| 추적 지수 | {benchmark} |
| 설정일 | {inception_date} |
| 보유 종목 수 | {holdings_count} |
| 상위 10종목 집중도 | {top10_pct}% |

**1.2 투표 결과 요약:**

3개월/6개월 Bull·Base·Bear 확률 + 합의 판정 (Strong/Moderate/Mixed)

**1.3 핵심 메시지 (3~5줄):**

가장 중요한 결론을 짧고 명확하게. 
예: "AI 인프라 투자 사이클의 수혜가 기대되나, 금리 불확실성으로 단기 상방은 제한적. 6개월 관점에서 Bull 우세(46%), 어닝 시즌이 분기점."

### 3. 에이전트 토론 하이라이트 (1~2페이지) ★차별점

**3.1 주요 시각 충돌:**

에이전트 간 가장 큰 의견 불일치를 2~3개 선정하여 서술.

예:
> **Fundamental vs Sentiment**: Fundamental Agent는 PER 15x가 5년 평균 대비 -1σ로 매력적이라 판단한 반면, Sentiment Agent는 Reddit 부정 센티먼트 급증(r/investing 부정률 65%)과 기관 자금 유출 3주 연속을 근거로 단기 하방 리스크를 경고. 교차 검토 후 Fundamental은 Bull 확률을 40%→35%로 하향 조정.

**3.2 합의 도달 과정:**

의견서 수정 이력, 어떤 근거에 의해 수정되었는지

**3.3 소수 의견:**

투표에서 진 의견도 명시적으로 수록. 
"X Agent는 [근거]를 이유로 [반대 의견]을 유지했다."

## XLSX 데이터 시트 구성

xlsx 스킬을 사용하여 생성. 별도 파일.

### Sheet 1: ETF Profile
- 기본 정보, 가격 시계열 (일별)

### Sheet 2: Holdings
- 전체 보유종목 리스트 + 섹터/지역 집계

### Sheet 3: Fundamentals
- 집계 밸류에이션, 성장, 수익성 지표

### Sheet 4: Performance
- 수익률, 리스크 지표, 롤링 분석

### Sheet 5: Sentiment Log
- 소스별 센티먼트 스코어 + 핵심 인용 + URL

### Sheet 6: Regime/Macro
- 국면별 성과, 매크로 상관계수

### Sheet 7: Peer Comparison
- 피어 비교 매트릭스

### Sheet 8: Vote Matrix
- 에이전트별 원본 의견서 + 수정 의견서 + 투표 결과

### Sheet 9: Debate Log
- 교차 검토 전문 (동의/반박/충돌/수정)

## 보고서 스타일 가이드

- **언어**: 한국어 (기관투자자 대상)
- **톤**: 전문적이되 간결. 불필요한 수식어 최소화.
- **숫자**: 소수점 2자리 (수익률, 비율), 정수 (확률, 순위)
- **테이블**: 항목이 5개 이상이면 테이블 사용
- **강조**: 핵심 수치는 볼드 (DOCX 서식)
- **출처**: 센티먼트 인용 시 반드시 소스 URL 또는 출처 명시
- **면책**: 표지와 투자관점 섹션에 면책 문구 포함
